# Volunteens

🌍 Helping students discover life-changing opportunities worldwide.

Volunteens is a platform that curates fully-funded scholarships, summer programs, competitions, research opportunities, internships, and exchange programs for ambitious students, especially those from Central Asia.

## Mission

Many talented students miss opportunities because information is scattered across the internet. Volunteens brings trusted opportunities into one place, making it easier for students to discover, organize, and apply for programs that can transform their future.

## Features

- 🎓 Verified scholarships and educational programs
- 🌎 Global opportunities with a focus on Central Asia
- 🔍 Powerful search and category filtering
- 📅 Deadline tracking
- 🔖 Bookmark and save favorite opportunities
- ✍️ Essay workspace for application preparation
- 🌐 Multi-language support (English, Uzbek, Russian)
- 📱 Responsive modern design

## Categories

- Scholarships
- Summer Programs
- Competitions
- Research Opportunities
- Internships
- Exchange Programs
